# Password Generator
A simple and secure password generator written in Python that creates 
random passwords with a mix of special characters, numbers, uppercase letters, 
and lowercase letters.

# Features
Customizable password length.
Includes special characters, digits, uppercase and lowercase letters.
Generates passwords with a mix of different character types for improved security.

# Requirements
Python 3.x

# Usage
Enter the desired password length when prompted.

# Example Output
Enter the length of password you want: 15
Your password is: a8#F$h2q9!eR

# Customization
You can modify the ranges for the number of special characters, digits, and uppercase letters within the code.
Adjust the character sets if you want to include or exclude specific characters.
